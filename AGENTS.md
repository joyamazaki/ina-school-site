# 個別指導イーナ Webサイト制作ルール

## 技術構成
- 静的HTML、CSS、JavaScriptで制作する
- GitHub Pagesで公開する
- 主なページは index.html、pricing.html、contact.html
- 不要なHTMLページを増やさない
- サーバー側処理や複雑なフレームワークは、必要性が明確になるまで導入しない

## デザイン
- 日本語・欧文ともにGen Interface JPを使用する
- font-weightは400（Regular）のみ使用する
- Medium、SemiBold、Bold、ExtraBold、Blackは禁止
- UIに使用する色は、紺 #45489a、グリーン #19a839、グレー #efefef、ホワイト #ffffff の4色のみとする
- 横長ロゴ（assets/logo-horizontal.svg）は支給された元データをそのまま使用し、色、形状、グラデーションを改変しない
- 横長ロゴ内の色とグラデーションは、UIの4色ルールおよび透明度ルールの対象外とする
- ヘッダーには支給された横長ロゴを使用し、フッターには右側の「個別指導イーナ」の文字だけを白にした専用版を使用する
- フッター専用版を作成・調整する場合も、支給された元の横長ロゴは変更しない
- ページ背景はグレー #efefef、本文を配置する面はホワイト #ffffff、基本文字は紺 #45489a で統一する
- 大見出しとセクション見出しは白い面に紺 #45489a の縦線を添え、紺文字で左揃えにする
- 本文と説明文は白い面の上に配置し、紺やグリーンの背景へ直接載せない
- 白いタブやカードは、比較項目、FAQ、問い合わせなど囲う意味がある箇所だけに使用し、すべてのセクションをタブ化しない
- 小見出しは白い面の上で紺文字を使い、グリーン #19a839 のラベルやアイコンを補助的に使用する
- グリーン #19a839 はLINE導線、英字ラベル、場所・サービスの補助アイコン、料金やFAQの補助情報に使用する
- 本文、注記、FAQの文字サイズは16px未満にしない
- 文字サイズは本文16px、小見出し20px、見出し32px、大見出し48pxの4段階を基準とする
- スマートフォンでは見出し30px、大見出し36pxまで縮小してよい
- シャドウに使用できる色はグレー #efefef のみとする
- グレー #efefef を基準色としたシャドウに限り透明度を使用してよい
- 背景、文字、ボタン、境界線など、シャドウ以外の透明度表現は禁止
- スクロール表示アニメーションは透明度を使わず、移動とクリップで表現する
- 日本語の途中に不自然な強制改行を入れない
- PCとスマートフォンの両方へ対応する
- カード全体をリンクにしない
- 「詳しく見る」ボタンを増やしすぎない
- ボタンや情報量を増やしすぎず、簡潔な構成を維持する
- 駒込hahaco校と笹塚校のGoogle Mapリンクを残す
